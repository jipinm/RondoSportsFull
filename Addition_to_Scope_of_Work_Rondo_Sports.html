2Markup Pricing Feature

New functionality to add and manage markup prices directly within the Tickets Module of your admin application.

What's Being Added:

  • Input field to enter markup value for each ticket
  • Support for both percentage (%) and fixed amount ($) markup
  • Save markup values in the database
  • View and edit existing markup values
  • Automatic calculation of final ticket price (base price + markup)

How It Works:

Action Description
Add Markup Admin enters markup percentage or fixed amount for a ticket
Save System saves the markup value associated with the ticket
Calculate Final price = Base Price + Markup (automatically calculated)
Display Updated price shown to customers

Benefits:

  • Easy control over ticket pricing and profit margins
  • Flexibility to adjust pricing strategies per ticket
  • Streamlined pricing management from admin panel
